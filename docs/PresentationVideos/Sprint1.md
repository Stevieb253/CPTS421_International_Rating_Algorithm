## Project Overview
The **International Student Scoring System** is designed to help the WSU admissions office predict which international applicants are most likely to enroll. The system automates scoring applicants based on multiple factors, reducing manual work and improving decision-making efficiency.

## How to Use
This Sprint 1 demo showcases the following features:

1. Uploading documents (transcripts, financials, essays)  
2. Calculating category-specific and overall applicant scores  
3. Flagging applications for manual review  
4. Exporting results into a spreadsheet  

*(Note: Advanced features such as fraud detection refinement will be implemented in future sprints.)*
