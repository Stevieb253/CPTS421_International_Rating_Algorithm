# WSU International Applicant Rating System - UI Mockups

## Overview
Interactive HTML mockup for WSU's International Applicant Rating System - a desktop application helping admissions staff evaluate and manage international student applications.

## How to Use
1. Save the code as `index.html`
2. Open in any modern web browser
3. Click page tabs to navigate between screens

## Screens Included
1. **Login** - Secure authentication
2. **Dashboard** - Stats overview, charts, notifications
3. **Applicants** - Searchable/filterable applicant list
4. **Detail View** - Complete applicant profile with tabs
5. **Agents** - Recruiting agent performance tracking
6. **Reports** - Data visualization and export options
7. **Settings** - Algorithm configuration and preferences

## Design Specs
- **Primary Color**: `#981e32` (WSU Crimson)
- **Font**: Segoe UI, sans-serif
- **Optimized for**: Desktop (1400px+)

## Key Features
- Color-coded quality ratings
- Advanced search and filtering
- Agent performance metrics
- Data export functionality
- Algorithm weight adjustment

## For Developers
Suggested stack: React/Vue.js, Tailwind CSS, Chart.js
