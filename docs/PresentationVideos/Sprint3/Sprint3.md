## Project Overview

The **International Student Scoring System** continues to support WSU’s admissions office by scoring international applicants more accurately and efficiently. In **Sprint 3**, we enhanced the system with additional analytics, improved fraud detection, and streamlined document handling to better assist admissions decisions.

## Sprint 3 Features

This Sprint 3 demo showcases the following enhancements:

1. **Enhanced fraud detection** – Improved algorithms to detect suspicious financial documents and transcripts.
2. **Batch document processing** – Upload, process, and score multiple applications simultaneously.
3. **Export/Import functionality** – Easily import CSV files and export scoring results for reporting.
4. **Improved user interface** – Streamlined dashboards for faster review and interaction.
