Upload Financial PDF(s)

Select one or more bank statements or financial support letters. The system will run OCR, quick image checks, and an AI analysis, then combine them.

What this does:
  • Runs OCR to extract text (may be slightly noisy).
  • Computes quick pixel heuristics (ELA, copy-move).
  • Uses GPT-based vision to flag unusual regions and content conflicts.
  • Combines everything into a conservative LOW / MEDIUM / HIGH risk label.
  • Up to 20 pages analyzed automatically per document.

Note: This is a pre-screening tool only. Humans must review any MEDIUM or HIGH results.
